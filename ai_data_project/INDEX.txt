HOW TO USE
═══════════════════════════════════════════════════════════════════════════════

1. UPLOAD DATA:
   → Go to home page: http://localhost:8000/
   → Upload CSV file
   → Click "Upload"

2. ACCESS ML SECTION:
   → Click "🤖 ML Models" in navbar
   → Or go directly to /ml/

3. TRAIN ALL MODELS:
   → Select target column
   → Click "Train All Models"
   → Wait for training to complete

4. VIEW RESULTS:
   → See comparison table
   → Best model highlighted (green)
   → Click model name for details

5. OPTIONAL - TRAIN SPECIFIC MODEL:
   → Go to /ml/train/
   → Select model and target column
   → View detailed results
