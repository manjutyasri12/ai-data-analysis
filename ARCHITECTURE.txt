                                    ↕
                          DATA PROCESSING FLOW
                                    ↕
┌─────────────────────────────────────────────────────────────────────────────┐
│                        DATA FLOW - AUTO-TRAIN                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  1. USER UPLOADS CSV                                                        │
│     ↓                                                                       │
│  2. DJANGO STORES IN SESSION                                               │
│     ↓                                                                       │
│  3. USER SELECTS TARGET COLUMN                                             │
│     ↓                                                                       │
│  4. DETECT PROBLEM TYPE                                                     │
│     ├─ Float dtype → Regression                                            │
│     └─ Few unique/object → Classification                                  │
│     ↓                                                                       │
│  5. LOAD DATA FROM SESSION                                                 │
│     ↓                                                                       │
│  6. PREPARE FEATURES & TARGET                                              │
│     ├─ Remove duplicates                                                   │
│     ├─ Handle missing values                                               │
│     └─ Create feature matrix X and target y                                │
│     ↓                                                                       │
│  7. TRAIN-TEST SPLIT (80/20)                                               │
│     ↓                                                                       │
│  8. BUILD PREPROCESSOR                                                      │
│     ├─ Numeric: Impute + Scale                                             │
│     └─ Categorical: Impute + OneHot                                        │
│     ↓                                                                       │
│  9. FOR EACH MODEL:                                                         │
│     ├─ Load model instance                                                 │
│     ├─ Create pipeline (preprocessor + model)                              │
│     ├─ Train on X_train, y_train                                           │
│     ├─ Predict on X_test                                                   │
│     ├─ Compute metrics                                                     │
│     └─ Store result                                                        │
│     ↓                                                                       │
│  10. FILTER VALID RESULTS                                                   │
│     ├─ Remove None accuracies                                              │
│     └─ Safe float conversion                                               │
│     ↓                                                                       │
│  11. SORT BY ACCURACY (DESCENDING)                                          │
│     ↓                                                                       │
│  12. FORMAT FOR DISPLAY                                                     │
│     ├─ Best model first                                                    │
│     ├─ Failed models at end                                                │
│     └─ All metrics included                                                │
│     ↓                                                                       │
│  13. RENDER COMPARISON TABLE                                               │
│     ├─ All 12 models shown                                                 │
│     ├─ Best highlighted (green)                                            │
│     ├─ All metrics visible                                                 │
│     └─ Expandable details                                                  │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│                      ERROR HANDLING ARCHITECTURE                            │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  Safe Float Conversion                                                      │
│  └─ def _safe_float(x):                                                     │
│     ├─ Return None if x is None                                            │
│     ├─ Convert to float                                                    │
│     ├─ Detect NaN/Inf → Return None                                        │
│     └─ Exception → Return None                                             │
│                                                                             │
│  Individual Model Errors                                                    │
│  └─ try:                                                                    │
│     ├─ Train model                                                         │
│     ├─ Compute metrics                                                     │
│     └─ catch Exception:                                                    │
│        └─ Mark as failed, continue with others                             │
│                                                                             │
│  Results Filtering                                                          │
│  └─ valid_results = [r for r in results if r['accuracy'] is not None]     │
│                                                                             │
│  Template Error Display                                                     │
│  └─ {% if result.error %}                                                  │
│     └─ Display error message to user                                       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│                    METRICS CALCULATION ARCHITECTURE                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  REGRESSION METRICS:                                                        │
│  ├─ MAE = mean_absolute_error(y_true, y_pred)                              │
│  ├─ MSE = mean_squared_error(y_true, y_pred)                               │
│  ├─ RMSE = sqrt(MSE)                                                       │
│  ├─ R² = r2_score(y_true, y_pred)                                          │
│  └─ Accuracy = R² (for regression)                                         │
│                                                                             │
│  CLASSIFICATION METRICS:                                                    │
│  ├─ Accuracy = accuracy_score(y_true, y_pred)                              │
│  ├─ Precision = precision_score(..., average='macro')                      │
│  ├─ Recall = recall_score(..., average='macro')                            │
│  └─ F1 = f1_score(..., average='macro')                                    │
│                                                                             │
│  All metrics:                                                               │
│  ├─ Safe float conversion                                                  │
│  ├─ NaN/Inf detection                                                      │
│  └─ None for invalid values                                                │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
